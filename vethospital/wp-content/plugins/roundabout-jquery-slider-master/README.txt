This plugin allows you to insert a standard version of the Roundabout jQuery rotating slider into any page or post using a shortcode.

== Description ==

This plugin is based on the Roundabout slider developed by Fred LeBlanc and integrates the standard carousel version of this slider into a handy wordpress plugin that allows you to easily insert images using shortcodes directly into any blog or post.  This plugin has highly configurable options for title and description styling along with custom image sizes automatically scaled by the plugin.

Roundabout v.1.1.5 Functions and Features:

* Ability to Easily Insert Roundabout Carousel Style Slider
* Highly Configurable Styling Options for Each Slider
* Automatic Image Re-sizing and Custom Image Size Options
* Easy To Use Shortcodes For Quick Insertion Of Sliders.


[Changelog](http://wordpress.org/extend/plugins/roundabout-jquery-slider/changelog/)

== Installation ==

1. Download the plugin from our website or from the WordPress Plugin Directory.
2. Navigate to Plugins->Add New->Upload and upload the plugin you downloaded.
3. Click activate, then you will see a WP RoundAbout icon on the left.
4. Your Plugin is now installed!
